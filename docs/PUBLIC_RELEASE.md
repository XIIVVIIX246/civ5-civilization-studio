# Public release information

## Unofficial fan tool

Civ V Civilization Studio is an unofficial fan-made modding tool. It is not
affiliated with, endorsed by, or supported by Firaxis Games, 2K, Take-Two
Interactive, Aspyr, or Valve. Sid Meier's Civilization and related marks are
trademarks of their respective owners.

Civilization V: Brave New World is required and is not included.

## Optional Promotions - Expansion Pack integration

Promotions - Expansion Pack was separately created by **Bloublou** and is not
bundled with Civilization Studio. If a civilization author enables this
optional integration, every player must separately subscribe to and enable the
mod from Steam Workshop:

<https://steamcommunity.com/sharedfiles/filedetails/?id=84863495>

Civilization Studio records the dependency in the generated mod and presents
read-only compatibility metadata. It does not redistribute Bloublou's mod,
gameplay XML, or art assets.

## Privacy

Civilization Studio is a local desktop application. It does not include
telemetry, advertising, accounts, or an automatic upload service. Projects,
source artwork, generated mods, diagnostics, and backups remain on the user's
computer unless the user chooses to share them.

Project files can contain the creator name, local file paths, imported artwork,
and mod content entered by the user. Authors should review generated projects
before posting them publicly.

## Support and testing boundary

Static validation cannot prove that a generated civilization works in the
game. Test the exact generated mod in Civilization V: Brave New World and, if
claimed compatible, with In-Game Editor enabled. Back up important projects
before upgrading or replacing installed mods.
