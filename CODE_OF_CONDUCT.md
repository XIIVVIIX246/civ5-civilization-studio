# Community code of conduct

This project welcomes people of every experience level, especially first-time
mod authors. Be respectful, patient, constructive, and specific.

Harassment, threats, discrimination, doxing, sexualized abuse, deliberate
misinformation, and repeated disruption are not acceptable. Critique code and
ideas without attacking people. Respect copyright, privacy, and the rules of
the communities where this project is discussed.

Project maintainers may edit or remove contributions and restrict participation
when behavior harms the community. Report conduct concerns privately to the
repository owner through GitHub. Enforcement decisions should consider
context, severity, recurrence, and community safety.
